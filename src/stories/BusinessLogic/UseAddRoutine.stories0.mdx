# useAddRoutine

El hook `useAddRoutine` se utiliza para gestionar el estado y la lógica necesaria para agregar una nueva rutina a un paciente. Este hook proporciona un modal con un formulario que permite a los usuarios ingresar los detalles de la rutina y seleccionar el paciente al cual se agregará la rutina.

## Ejemplo de implementación

```typescript
import useAddRoutine from '../../hooks/useAddRoutine';
import React from 'react';
import { Modal, Button } from 'react-bootstrap';

const AddRoutineComponent = () => {
    const { 
        showModalAddRoutine, 
        toggleModalAddRoutine, 
        modalTitleAddRoutine, 
        modalContentAddRoutine 
    } = useAddRoutine();

    return (
        <div>
            <Button onClick={toggleModalAddRoutine}>Agregar Rutina</Button>
            <Modal show={showModalAddRoutine} onHide={toggleModalAddRoutine}>
                <Modal.Header closeButton>
                    <Modal.Title>{modalTitleAddRoutine}</Modal.Title>
                </Modal.Header>
                <Modal.Body>
                    {modalContentAddRoutine}
                </Modal.Body>
                <Modal.Footer>
                    <Button variant="secondary" onClick={toggleModalAddRoutine}>
                        Cerrar
                    </Button>
                    <Button variant="primary" type="submit">
                        Guardar Cambios
                    </Button>
                </Modal.Footer>
            </Modal>
        </div>
    );
};

export default AddRoutineComponent;
```

## Descripción de los elementos del hook

### Estados

**showModalAddRoutine**

- Tipo: boolean
- Descripción: Estado que indica si el modal para agregar una rutina está visible o no.

**AddRoutineData**

- Tipo: AddRoutineDataTypes | undefined
- Descripción: Objeto que contiene los datos de la rutina que se va a agregar.

### Métodos

**toggleModalAddRoutine**

- Tipo: () => void
- Descripción: Función que alterna la visibilidad del modal para agregar una rutina.

### Constantes

**modalTitleAddRoutine**

- Tipo: string
- Valor: 'Agregar Rutina'
- Descripción: Título del modal para agregar una rutina.

**modalContentAddRoutine**

- Tipo: JSX.Element
- Descripción: Contenido del modal, que incluye un formulario con los campos necesarios para agregar una rutina.


### Dependencias
Para usar este hook, necesitas instalar las siguientes dependencias:
- `npm install react-bootstrap`

### Notas
- Asegúrate de que los datos de los pacientes estén correctamente configurados en el estado global para que el hook funcione correctamente.
- La función setAddRoutineData actualiza el estado del formulario a medida que el usuario ingresa datos.
- El formulario está configurado para cambiar dinámicamente los datos de la rutina basada en la entrada del usuario.
