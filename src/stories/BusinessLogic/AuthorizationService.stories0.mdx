# AuthorizationService

El servicio `AuthorizationService` se utiliza para gestionar el token de autorización del usuario. Este servicio proporciona métodos para establecer y eliminar el token en el almacenamiento local.

# Ejemplo de implementación

```typescript
import AuthorizationService from '../../services/AuthorizationService';

// Establecer un token
AuthorizationService.setToken('nuevoToken');

// Eliminar el token
AuthorizationService.removeToken();
```

## Métodos

### setToken:

```typescript

async function setToken(newToken: string) {
    const token = `${newToken}`
    window.localStorage.setItem('token', token)
}

```
    - Descripción: Establece un nuevo token de autorización en el almacenamiento local.
    - Parámetros:
        - **`newToken`**: El nuevo token de autorización que se debe establecer.
    - Uso: `AuthorizationService.setToken('nuevoToken');`

### removeToken:
```typescript

async function removeToken() {
    window.localStorage.removeItem('token')
}

```

    - Descripción: Elimina el token de autorización del almacenamiento local.
    - Uso: `AuthorizationService.removeToken();`

## Notas
- Asegúrate de que el token se maneje de manera segura y que no se exponga en lugares no seguros.
- El almacenamiento local se utiliza para almacenar el token de autorización, lo cual puede no ser adecuado para todos los tipos de aplicaciones, especialmente aquellas que manejan datos sensibles.
